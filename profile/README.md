# OpenCourt WebApp

OpenCourt is a web app for court scheduling and management.

## Stack do projeto

### Frontend

Next.js

JavaScript

Tailwind CSS

shadcn/ui para componentes

----

### Backend

Next.js Route Handlers / Server Actions

Prisma como ORM

PostgreSQL como banco de dados

Supabase para banco e autenticação

----

### Deploy

Vercel para hospedar a aplicação

Supabase para infraestrutura de banco e auth
